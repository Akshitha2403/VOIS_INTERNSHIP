🔐 VOIS Internship Project

⚡ Nmap Port Scanner with Modern GUI

📌 Overview

This project is a GUI-based Port Scanner developed using Python.
It allows users to scan a target system for open ports in a simple and user-friendly interface.
The application uses socket programming and multithreading to perform fast and efficient port scanning.

🚀 Features

* Modern Dark-Themed GUI using CustomTkinter
* Scan any IP address or hostname
* Supports custom port range
* Displays open ports with services
* Multi-threaded scanning for better performance
* Option to clear and save results

🛠️ Technologies Used

* Python
* CustomTkinter (GUI)
* Socket Programming
* Multithreading

▶️ How to Run

1. Clone the repository:

[https://github.com/Akshitha2403/VOIS_INTERNSHIP.git]

2. Navigate to the project folder:

cd VOIS_INTERNSHIP

3. Install dependencies
   
pip install customtkinter

4. Run the application:

python portscannergui.py

📷 Output

The application provides a clean interface where users can:

* Enter target IP
* Specify port range
* View scan results in real-time
